### Comandos mais utilizados:

* (* ) asterisco com espaço - cria um bullet point (lista desordenada)
* **negrito** (** negrito **) - torna a palavra em negrito
*  (#) - Varia de acordo dos os tamanhos de texto do html de h1(um #) até h6 ( seis #)
